bleeter¶

Warning

This documentation was built from a Git tree, and may not correspond directly to any released version. Check the release tarballs for documentation for a particular version of bleeter.

bleeter is a nasty little viewer for twitter, currently very much in a Works For Me state. It isn’t intended to be used by others, but perhaps others will find it useful.

All it does is fetch your friends timeline and display notification popups for new tweets. If your system’s notification daemon supports adding actions [1] to the popups then you’ll be able to make a number of choices by clicking the buttons on the popup.

[1]Most common notification daemons support actions, including xfce4-notifyd and Galago’s notification-daemon.

It is written in Python, and requires v2.6 or later. bleeter is released under the GPL v3
