InstallationÂ¶

You can download releases from the downloads page.

bleeter requires Python v2.6(or newer) and the following Python modules:

  • libnotify Python bindings
  • configobj
  • pygtk
  • tweepy

The following optional packages will be used if available:

  • setproctitle for displaying bleeter state in process title
  • termcolor for coloured output in the terminal

If you’re using Gentoo all the dependencies are available from either the main tree or my overlay.

bleeter doesn’t require installation to work as it is only a single Python script.
